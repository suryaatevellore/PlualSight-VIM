Using Vim elsewhere

Once you've gotten used to using Vim, you'll wish you could use its motions elsewhere. When we talked about installation, we mentioned configuring your shell to use Vim as its EDITOR. You can also tell it you want to use Vim keybindings like this:

# In your ~/.inputrc
set editing-mode vi

# In your ~/.editrc
bind -v

This should spread the good news beyond your shell to readline, the mysql client, irb, and so on.

Terminal Vim also works very well with Screen. We won't talk about it here, but if you like one you'll probably like the other.

MacVim provides a few OS X system services, and can also act as an external editor for OS X apps which support the ODB Editor Protocol, such as Mail.app.

Finally, Vimium is a Chrome extension which gives you a lot of Vim keybindings for navigating the browser page, manipulating tabs, and jumping through your browsing history. It also supports command repetition and mappings. It's well worth taking for a spin.
